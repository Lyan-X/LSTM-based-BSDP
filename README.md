# BSDP
基于 LSTM 的共享单车小时级需求预测项目，涵盖数据预处理、LSTM 模型训练、Django Web 预测系统部署的完整流程。技术栈：Python+TensorFlow+Django+Pandas/Numpy。包含可复现代码、环境配置指南、踩坑解决方案，适合机器学习新手入门时间序列预测，也可作为本科毕设参考项目。Bike sharing hourly demand prediction project based on LSTM, covering the full workflow of data preprocessing, LSTM model training, and Django web prediction system deployment. Tech stack: Python+TensorFlow+Django+Pandas/Numpy. Includes reproducible code, environment setup guide, and troubleshooting solutions. Suitable for ML beginners to get started with time series prediction, and can be used as a reference for undergrad thesis.
